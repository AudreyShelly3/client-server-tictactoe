# client-server-tictactoe
This project is a multiplayer Tic-Tac-Toe web application built using Blazor and SignalR. It demonstrates real-time communication between clients and a server, allowing players to compete in an interactive game environment.

---

## Overview

The application uses a client-server architecture where multiple users can connect and play Tic-Tac-Toe in real time. SignalR enables live communication between players, while SQLite is used to store game data and user statistics.

---

## Features

• Multiplayer Tic-Tac-Toe gameplay  
• Real-time updates using SignalR  
• Client-server architecture  
• Persistent data storage with SQLite  
• User interface built with Blazor

## How to Run the Project

1. Clone the repository

```
git clone https://github.com/AudreyShelly3/client-server-tictactoe.git
```

2. Open the project in Visual Studio or Visual Studio Code.

3. Restore dependencies.

4. Run the project.

5. Open the application in your browser and start a multiplayer game.
